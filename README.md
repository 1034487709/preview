preview
=======

纯浏览器端的图片预览<br/>
支持IE5.5~IE11、Chrome、FF、Safari和Opera<br/>

**v0.4**<br/>
新增上传文件MIME类型筛选。默认值为`image/*`,通过input的accept属性值设置。<br/>

**v0.3**<br/>
修复FF3.0不支持`FileReader`的bug。<br/>

**v0.2**<br/>
修复IE11下当`document.documentMode < 10`时无法预览图片的bug。<br/>

**v0.1**<br/>
通过滤镜和`FileReader`实现图片预览功能。<br/>
